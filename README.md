# f5prng

Pure-Go, password-seeded PRNG for F5 steganography, compatible with Java's SHA1PRNG. It is deterministic and **not cryptographically secure** — use `crypto/rand` when you need real randomness.

## Install

```bash
go get github.com/0verkilll/f5prng
```
